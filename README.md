# 📊 Dashboard de Funcionários - Power BI  

Este projeto é um **dashboard interativo** criado no **Power BI** para visualizar e analisar os dados dos funcionários de uma empresa fictícia. Os dados foram inicialmente armazenados em um arquivo **Excel** e depois transferidos e manipulados no Power BI, utilizando **DAX** e técnicas de limpeza e transformação de dados.

## 🚀 Funcionalidades  

✅ **Visão geral da equipe**: total de funcionários, média salarial e distribuição por departamento.  
✅ **Análises detalhadas**: gráficos de cargos e setores.  
✅ **Filtros interativos**: para explorar dados específicos de forma dinâmica.  
✅ **Métricas personalizadas**: criadas com **DAX** para cálculos avançados.  

## 🛠️ Tecnologias utilizadas  

- **Power BI** (visualização e análise de dados)  
- **DAX** (medidas e cálculos personalizados)  
- **Excel** (armazenamento e manipulação inicial dos dados)
